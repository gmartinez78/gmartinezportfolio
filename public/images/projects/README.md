Recommended subfolders inside each project:

- `hero/` for page hero images
- `thumbnails/` for home/projects card thumbnails
- `banners/` for section banners and showcase art
- `logos/` for project-specific brand/client logos
- `gallery/` for supporting screenshots and panels
- `misc/` for one-off assets that do not fit the categories above

Notes:

- Existing live assets are still referenced from the current shared paths.
